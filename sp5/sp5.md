# Llicències de Windows: què són i quant costen

## Tipus principals de llicència

### Windows Home

Pensada per a ús domèstic

Inclou:

* Microsoft Store
* Gaming i Xbox integration
* BitLocker bàsic (segons versió)
* Funcions normals d’usuari

No inclou:

* Eines avançades d’empresa
* Hyper-V complet
* Gestió de dominis

Ideal per:

* Portàtils personals
* Gaming
* Estudiants
* Ús quotidià

---

### Windows Pro

Versió orientada a professionals i empreses

Afegeix:

* BitLocker complet
* Remote Desktop host
* Hyper-V
* Unió a dominis
* Polítiques de seguretat avançades
* Windows Sandbox

Ideal per:

* Feina professional
* Programació
* Empreses
* Virtualització

---

### Windows Enterprise

Per a grans empreses

Inclou:

* Seguretat corporativa avançada
* Gestió massiva de dispositius
* Control IT centralitzat
* Funcions enterprise exclusives

Normalment no es compra individualment.

---

# Tipus de clau / llicència

| Tipus           | Característiques                                  |
| --------------- | ------------------------------------------------- |
| Retail          | Oficial, transferible entre PCs                   |
| OEM             | Més barata, queda vinculada a una sola placa base |
| Volume / MAK    | Empreses, no pensada per a particulars            |
| Digital License | Activació associada al compte Microsoft           |
