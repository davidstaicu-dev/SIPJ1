# Llicències de Windows Server

## Què és Windows Server?

Windows Server 2025 és la versió de Windows pensada per:

* servidors
* empreses
* virtualització
* Active Directory
* fitxers compartits
* Hyper-V
* serveis remots
* hosting intern

A diferència de Windows normal, està preparat per treballar 24/7 i gestionar molts usuaris.

---

# Edicions principals

## Essentials

Pensada per a petites empreses.

Limitacions:

* fins a 25 usuaris
* 50 dispositius
* màxim 10 cores
* 1 CPU
* 128 GB RAM

Avantatges:

* més simple
* no necessita CALs separades
* barata

Ideal per:

* petites oficines
* NAS avançat
* servidor de fitxers

⚠️ Des de Server 2022/2025, Essentials és pràcticament una llicència especial basada en Standard. ([software-express.de][1])

---

## Standard

La més utilitzada.

Inclou:

* Active Directory
* Hyper-V
* serveis empresarials
* 2 màquines virtuals incloses
* Docker / contenidors

Necessita:

* llicència per cores
* CALs d’usuari o dispositiu

Ideal per:

* empreses normals
* virtualització moderada
* servidors interns

---

## Datacenter

La versió “bestial”.

Inclou:

* virtualització il·limitada
* funcions avançades de xarxa
* Storage Spaces Direct
* Shielded VMs
* entorns cloud

Ideal per:

* datacenters
* moltes VMs
* hosting
* infraestructura gran

---

# Diferències ràpides

| Característica      | Essentials | Standard | Datacenter   |
| ------------------- | ---------- | -------- | ------------ |
| Petita empresa      | ✅          | ✅        | ⚠️           |
| Active Directory    | ✅          | ✅        | ✅            |
| Hyper-V             | Limitat    | ✅        | ✅            |
| VMs incloses        | 1          | 2        | Il·limitades |
| CALs necessàries    | ❌          | ✅        | ✅            |
| Virtualització gran | ❌          | ⚠️       | ✅            |
